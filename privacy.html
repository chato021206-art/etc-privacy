プライバシーポリシー

本プライバシーポリシー (以下「本ポリシー」といいます) は、Hirokazu Kataoka (以下「当方」といいます) が提供する iOS アプリ「ETC カレンダー」 (以下「本アプリ」といいます) における、利用者の個人情報および 利用者情報の取扱いについて定めるものです。

本アプリの基本方針
利用者が本アプリへ取り込んだ ETC 利用明細、勤務記録、給与・手当などの記録、 および NEXCO ETC 利用照会サービスの認証情報そのものは、当方サーバーには 送らず、利用者の iOS 端末内、iOS Keychain、または利用者の iCloud Drive に 保存されます。
ただし、距離・所要時間・出発時刻を自動計算する機能では、利用者が入力した 出発地・目的地 (自宅住所を含みます)・到着希望時刻・日付を、 当方が運用するプロキシ (https://traffic-calendar.pages.dev/api/distance) 経由で Google の経路 API へ送信します。

1. はじめに

本ポリシーは、本アプリが取り扱う個人情報の種類、利用目的、管理方法、 第三者提供の有無等について明らかにすることで、 利用者の個人情報に対する適切な取扱いを行うことを目的としています。

本アプリの利用にあたっては、本ポリシーをご確認のうえ、内容に同意いただいた ものとして取り扱います。同意いただけない場合は、本アプリの利用をお控えください。

2. 取得する情報

本アプリは、以下の情報を取得します。 保存場所は原則として利用者ご自身の端末内または利用者の iCloud Drive です。 経路計算機能で外部へ送信する情報については、3 章および 4 章に記載します。

項目内容保存場所
ETC 利用明細データ 日時、入口・出口 IC (インターチェンジ)、通行料金、車種、車両番号末尾等、利用者が CSV / PDF ファイルから本アプリへ取り込んだ明細情報 本アプリのローカルストレージ (端末内)
タグ情報・分類情報 利用明細に対して利用者が付与した「仕事」「私用」等の分類情報 本アプリのローカルストレージ (端末内)
アプリ設定情報 表示設定、通知設定等、利用者が本アプリ内で行った設定情報 本アプリのローカルストレージ (端末内)
勤務シフト・給与/手当などの金額 勤務日、日勤・夜勤などの区分、通勤距離、給与・手当・交通費など、利用者が入力または本アプリ内で計算した勤務関連情報 本アプリのローカルストレージ (端末内)
自宅住所・よく行く場所 (地点情報) 距離・所要時間・出発時刻の計算やルート表示のために、利用者が入力した自宅住所、勤務地、訪問先等の地点情報 本アプリのローカルストレージ (端末内)
NEXCO 認証情報 利用者が任意で保存した NEXCO ETC 利用照会サービスのログイン ID / パスワード iOS Keychain (端末内)

本アプリは、広告目的の位置追跡、連絡先、写真、生体情報、広告識別子等の 取得を目的とした機能を備えていません。ただし、経路計算のために、利用者が 入力した自宅住所や行き先などの住所・地点情報を取り扱います。

3. 利用目的

取得した情報は、以下の目的のためにのみ利用します。

  • 利用者が取り込んだ ETC 利用明細をカレンダー形式で表示・閲覧するため
  • NEXCO ETC 利用照会サービスから利用者の明細を自動取得するため
  • 利用明細の集計・分類 (経費精算等の用途を想定) を行うため
  • 勤務シフト、通勤距離、給与・手当などの金額を記録・集計するため
  • 距離・所要時間・出発時刻の自動計算のため、出発地・目的地・到着希望時刻を外部の経路計算サービスへ送信するため
  • 利用者が選択した形式で、利用明細データをエクスポート・バックアップ・復元するため
  • 利用明細の取込完了等を、利用者の端末内で通知するため

当方は、上記の目的の範囲で本アプリを提供します。外部送信を伴う機能の 詳細は、次章に記載します。

4. 第三者提供 / 外部送信

当方は、利用者の ETC 明細、勤務記録、NEXCO 認証情報そのものを当方サーバーで 保管して第三者へ提供することはありません。ただし、本アプリの機能上、 以下の外部サービスとの通信またはデータ移転が発生します。

  • 地図・経路計算 (Google): 距離・所要時間・出発時刻の計算のため、出発地・目的地 (自宅住所を含みます)・到着希望時刻・日付を、当方が運用するプロキシ (https://traffic-calendar.pages.dev/api/distance) 経由で Google の経路 API へ送信します。この送信は、利用者が地図アプリを 開く操作をした場合だけでなく、距離や帰り時間を自動計算する場合にも 発生します。このプロキシでは、通信ログ等が記録される場合があります。 Google における情報の取扱いは、Google のプライバシーポリシーに従います。
  • NEXCO ETC 利用照会サービス: 本アプリは、利用者が保存した認証情報を使って https://www2.etc-meisai.jp へ HTTPS で接続し、利用明細を取得します。NEXCO 認証情報や取得した明細は、 当方サーバーには送られません。 NEXCO サイト上での情報の取扱いについては、NEXCO の利用規約およびプライバシー ポリシーに従います。
  • Apple iCloud Drive: 利用者が「iCloud バックアップ」操作を行った場合、利用者ご自身の iCloud に バックアップファイルが保存されます。当方は当該ファイルにアクセスできません。 iCloud の取扱いは Apple のプライバシーポリシーに従います。
  • ファイル書き出し / 読み込み: 利用者が CSV / PDF / JSON 等の書き出し・取込操作を行った場合、利用者が 選択した保存先 (端末内、共有フォルダ、メール添付等) との間でファイルが やり取りされます。保存先のサービスの取扱いは、当該サービスの規約に従います。

5. 個人情報の管理

本アプリにおける情報の管理体制は以下のとおりです。

  • NEXCO ETC 利用照会サービスの認証情報 (ID / パスワード) は、 利用者が任意で保存した場合に限り iOS Keychain に保存されます。 認証情報は当方サーバーへ送信せず、開発者個人がその内容を閲覧することは ありません。ただし、本アプリは NEXCO 自動ログインのため、端末内の 認証情報を読み出して使用します。保存した認証情報はアプリから削除できます。
  • ETC 利用明細データ、勤務記録、自宅住所、よく行く場所、各種設定は、 本アプリのローカルストレージ内に保存されます。利用者が iCloud バックアップや ファイル書き出しを行った場合は、利用者が選択した保存先に保存されます。
  • NEXCO 自動取得は、初回設定時に過去 15 ヶ月分をまとめて取得します。 その後はアプリ起動時および前面復帰時に当月分を更新します。 ログイン失敗等が連続した場合は自動取得を停止し、利用者が再試行または 認証情報を更新するまで再開しません。
  • 経路計算に必要な出発地・目的地・到着希望時刻・日付は、HTTPS 通信で 当方のプロキシおよび Google の経路 API へ送信されます。プロキシ側の ログや保持仕様は、運用状況に応じて見直す場合があります。
  • 本アプリでは、iOS Keychain、HTTPS 通信、端末内ローカル保存を用いて 情報を管理します。情報の取扱いに関するお問い合わせは、11 章の窓口まで ご連絡ください。
  • 端末の紛失・盗難・故障により情報が失われた場合に備え、 利用者ご自身でバックアップを取得することを推奨します。 当方は、紛失等により失われたデータの復旧責任を負いません。

6. 個人情報の開示・訂正・削除

本アプリのローカルストレージ、iOS Keychain、または利用者の iCloud Drive に 保存された情報は、利用者ご自身の操作により閲覧・訂正・削除できます。

  • 閲覧・訂正: 本アプリ内の各画面 (カレンダー表示、設定画面等) から行えます。
  • 利用明細・タグ情報の削除: 本アプリ内の対応する画面から個別削除、または一括削除を行えます。
  • 全データの削除 (アプリ内): 本アプリの「設定」画面に 「全データを削除」 ボタンを用意しています。 これをタップすると、端末内の ETC 利用明細・タグ・保存済 PDF・NEXCO 取得キャッシュを削除できます。 カード名や表示・通知などの設定情報は残ります。
  • アンインストールによる削除: 本アプリを iOS 端末からアンインストール (削除) いただくことでも、 端末内に保存されたすべての情報が削除されます。 iCloud バックアップを取得済みの場合は、iCloud 側のファイル削除も 別途お願いします。

当方サーバーでは ETC 明細、勤務記録、NEXCO 認証情報そのものを保管していないため、 これらについて当方から提供できる情報はございません。経路計算の外部送信に関する お問い合わせは、11 章の窓口までご連絡ください。

7. クッキーおよび類似技術について

当方は、本アプリにおいて第三者提供の解析ツール (Google Analytics 等)、広告 SDK、トラッキング SDK 等を使用していません。 本アプリが NEXCO ETC 利用照会サービスへ接続する場合も、用途は利用者の明細取得に 限られ、広告目的のトラッキングには使用しません。

8. 子どもの個人情報について

本アプリは、ETC 利用明細を扱う性質上、13 歳未満の方の利用を想定しておりません。 13 歳未満の方は、保護者の同意のもとで利用してください。
